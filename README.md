# UUPM Clone - Professional Demo Website Platform

A comprehensive platform showcasing 20 original, professionally designed demo websites built with Next.js and React.

## Features

- ✨ **20 Original Demo Websites** with unique designs
- 📱 **Fully Responsive** - optimized for all device sizes
- 🎨 **Modern UI** with Tailwind CSS
- 🚀 **Performance Optimized** with Next.js
- 🧩 **Reusable Components** for consistency
- 📊 **Responsive Grid** showcase on homepage
- ♿ **Accessible** markup and navigation

## Getting Started

### Prerequisites
- Node.js 16.x or higher
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/hamzajavedgujjar/uupm-clone.git
cd uupm-clone

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
